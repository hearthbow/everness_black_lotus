# everness_black_lotus
Luanti/Minetest add-on mod for everness mod. Adds black version of white lotus flower, created by placing white lotus flower on everness cursed dirt.
Addon mod for everness mod by SaKeL on Luanti (formerly Minetest).

Adds a lotus flower variant, black version of the white lotus flower from everness. Created by placing white lotus flower on everness cursed dirt and glows at night.

# Features
Black lotus variant of the white lotus
Glows at night with animated effect like original lotus flowers from everness mod. 
Can be created by placing white lotus flower on everness cursed dirt.
Fully integrated with Everness mechanics

# Dependencies
everness
